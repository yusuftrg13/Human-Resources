# Human Resources Employee Report

## 📝 Project Overview
This project involves creating an interactive **HR Employee Dashboard** using Power BI. The dashboard visualizes key HR metrics such as employee turnover, work-life balance, average monthly income, and marital status distribution.

## 📊 Key Features
- **Total Monthly Income** and **Average Monthly Income** KPIs.
- Employee distribution by **Work-Life Balance** and **Business Travel**.
- **Marital Status** analysis with a detailed pie chart.
- **Department-based Average Revenue** visualization.
- Turnover analysis segmented by departments.

## 🔧 Tools Used
- **Power BI** for data visualization and analysis.
- **DAX (Data Analysis Expressions)** for advanced calculations.
  
## 🚀 How to Use
1. Download the `.pbix` file.
2. Open it in Power BI Desktop.
3. Interact with the visuals and explore the data.

# İnsan Kaynakları Çalışan Raporu

## 📝 Proje Hakkında
Bu proje, **Power BI** kullanılarak etkileşimli bir **İnsan Kaynakları Çalışan Dashboard'u** oluşturmayı amaçlamaktadır. Dashboard, çalışanlarla ilgili temel metrikleri (çalışan devir oranı, iş-yaşam dengesi, ortalama gelir, medeni durum dağılımı vb.) görselleştirmektedir.

## 📊 Öne Çıkan Özellikler
- **Toplam Aylık Gelir (Total Monthly Income)** ve **Departman Bazlı Ortalama Gelir (Average Monthly Income)** KPI'ları.
- **İş-Yaşam Dengesi (Work Life Balance)** ve **Seyahat Sıklığı (Business Travel)** bazında çalışan dağılımı.
- **Medeni Durum Analizi (Marital Status Analysis)** için pasta grafiği.
- **Departman Bazında Ortalama Gelir (Average Revenue by Department)** görselleştirmesi.
- Departmanlara göre **işten ayrılma oranları (Turnover)** analizi.

## 🔧 Kullanılan Araçlar
- **Power BI**: Veri görselleştirme ve analitik için.
- **DAX (Data Analysis Expressions)**: Gelişmiş hesaplamalar için.

- ## 🚀 Nasıl Kullanılır?
1. `.pbix` dosyasını indirin.
2. Power BI Desktop ile açın.
3. Görselleştirmelerle etkileşim kurarak veriyi keşfedin.
